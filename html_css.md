**2.padding:5px 15px 5px 15px; How to recognize it?**
  ```
  border 视为盒子的外壳，padding则视为盒子中的缓冲填充物，比如泡沫，那么剩下的就是盒子真正可以放东西的空间，比如里面的鞋子.
  margin 你可以理解为这个盒子和其他盒子之间的间隔，距离  
  
  padding:5px 15px 5px 15px; --->  上 右 下 左  (顺时针方向)
  ```
